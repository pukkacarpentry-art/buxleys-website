BUXLEYS SIMPLE STATIC SITE

This version does NOT need npm, Vite or a build step.

How to use with Vercel:
1. Download and unzip the file.
2. Upload the unzipped folder to a new GitHub repo.
3. Import that repo into Vercel.
4. In Vercel, set Framework Preset to 'Other' or leave it as no framework.
5. There is no build command needed.
6. Deploy.

Files:
- index.html
- style.css

If Vercel asks for:
- Build command: leave blank
- Output directory: leave blank

This is the simplest version possible.
